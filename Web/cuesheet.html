CueFuse - How to use it

An Example CUE Sheet

An example CUE Sheet from my CD collection is given below. Note the way IMAGE, DATE and GENRE have been added to the CUE Sheet semantics. MP3CueFuse understands these 'CUE Sheet' tags.

In the given example I have misused the PERFORMER tag per track to hold the concert part. However, MP3CueFuse will negate the performer field of tracks. Only the TITLE fields are used. PERFORMER and TITLE of the cuesheet itself are put into the '_.mp3' file as also the IMAGE, DATE and GENRE fields are.

Note: use e.g. EasyTag to add album art to your original MP3 file. The IMAGE tag is understood by Mp3CueFuse, but the image it isn't added to the tracks. The original album art of the MP3 file is. 
REM GENRE Classical
REM DATE 1985
REM IMAGE vierjaargetijden.jpg
PERFORMER "Academy of St. Martin in the Fields"
TITLE "The Four Seasons (Neville Marriner)"
FILE "Academy of St. Martin in the Fields - The Four Seasons (Neville Marriner).mp3" WAVE
